# 🛒 Ecommerce Sales Intelligence Project

## 📌 Overview
An end to end data analytics project analyzing 9994 ecommerce orders 
to uncover revenue leakage, profit trends and business insights.

## 🛠️ Tools Used
- **Python** (Pandas) — Data cleaning & feature engineering
- **MySQL** — Business analysis using SQL queries
- **Power BI** — Interactive dashboard

## 📁 Project Structure
| File | Description |
|------|-------------|
| Week1_Cleaning.ipynb | Data cleaning & feature engineering |
| Week2_Sql.ipynb | 9 SQL business queries |
| superstore_clean.csv | Cleaned dataset |
| new project.pbix | Power BI dashboard |

## 🔍 Key Insights Found
- 📦 **300+ products** selling but losing money
- 🪑 **Tables sub-category** has -14.77% profit margin despite ₹206K in sales
- 📍 **California + New York** generate 50% of total profit
- 💸 Products with **40%+ discount** have margins as low as -50%
- 🚚 **Standard Class** shipping used by 60% of customers but takes 5 days
